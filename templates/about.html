About NoteGlow

Why is sharing notes so hard and slow?

It doesn't have to be, so I made. Yet. Another. Note App. NoteGlow is simple note-sharing tool designed to help you stay productive without the hassle.

Pricing? Free?

Free! And it will stay free for the features you use today. I do have some advanced features in mind that might cost more to host, but those are for another day when I can take a break from my consultancy work. Oh, and if you’re into self-hosting, you’re welcome to grab a copy at GitHub and host NoteGlow yourself—free forever. If you love NoteGlow already, please support and you'll get Premium lifetime as soon as it is built

What's it made with?

Lots of OSS, but kept simple Honorable mentions:
  • Django is used for a base
  • Nash is an insanely simple WYSIWYG that has been customized for this project. I never thought I would enjoy integrating an WYSIWYG editor, but Nash is just so simple and nice.
  • Ubuntu is loved and hated by the OSS community, but is so nice and easy to work with and this is what is running NoteGlow.com

FORK ME or help a little

NoteGlow is simple, and I intend to keep it that way. But that’s what makes it the perfect place to contribute! The first version was built in less than 5 hours and is easy to understand and update. Whether you’re fixing bugs, suggesting features, or improving the docs, your help can make NoteGlow even better. Let’s build something great together! NoteGlow on Github
